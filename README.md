# Southview Piano - Web Application
Southview Piano is a local piano dealer.
#### Requirements:
1. `.NET 7`
2. `Visual Studio`

#### Development environment setup instructions:
1. Clone the repository on your local machine.
2. Open the root folder in `Visual Studio`.
